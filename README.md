## Features

- 현재 상영중인 영화 정보 Preview
- Trending 영화, TV 프로그램 무한스크롤
- 영화, TV 프로그램 정보 자세히 보기
- 영화, TV 프로그램 MyList 에 담기, 빼기
- MyList 조회
- 영화, TV 프로그램 검색
